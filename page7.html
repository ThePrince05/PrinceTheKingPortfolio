ShowStasher

A desktop WPF application that helps you organize, rename, and categorize your collection of movies, TV shows, and anime. ShowStasher fetches metadata from TMDb and Jikan, renames files accordingly, groups them into folders, and caches metadata locally for offline access.

Chat App

Chat App Using Supabase

This chat app is a collaborative project built using WPF and Supabase, with a focus on creating an engaging group chat application. Users can create and join groups for seamless real-time communication. The app features customizable user profiles, notifications, and search functionality. It runs on .NET 8.0 and leverages SQLite as its database. To set up the app, users need to create a Supabase account and follow a simple installation process.
