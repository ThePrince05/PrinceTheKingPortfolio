Comic Books 

Comic Books Web App

A modest web application, built with ASP.NET MVC, designed to cater to small businesses seeking a simple online presence. This application offers essential features such as a Home page, a Location page to display nearby comic book shops, a Favorites page showcasing curated recommendations, an About Us page detailing the shop's history, and a News page highlighting the latest comic book industry news. As my inaugural web application, this project holds a special place in my development journey.
